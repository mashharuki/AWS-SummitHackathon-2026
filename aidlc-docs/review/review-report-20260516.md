🔴 重大問題3：MVPスコープが予選14日間で実現不可能

現在のMVPスコープ（v1.0.0）:
- Slack Webhook + Gmail + Google Calendar の3サービス同時連携
- Bedrock AgentCore（新興技術・ドキュメント薄い）
- DynamoDB 8テーブル設計
- 14 REST APIエンドポイント
- SSEストリーミング表示
- Cognito + Google OAuth

これを14日で実装しデプロイするのは、経験豊富なフルスタックエンジニアでも困難です。

特に危険な組み合わせ:
1. Bedrock AgentCore: 2026年時点で新興技術。ハッカソン期間中に詰まった場合の代替案が設計されていない
2. Gmail API: OAuth スコープが複雑。審査には2〜3週間かかることがある（本番利用の場合）
3. 3サービス同時連携: 各サービスで別々のOAuthフローとWebhook設定が必要
