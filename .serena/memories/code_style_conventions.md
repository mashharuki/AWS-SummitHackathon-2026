# コードスタイルと規約

## 一般規約
- コードレビューは日本語（CodeRabbit設定より）
- コミットメッセージ: Conventional Commits形式 (`feat:`, `fix:`, `docs:` 等)

## AI-DLCプロセス規約
- AGENTS.md を最初に読むこと（最高優先度）
- 各フェーズ完了前にユーザー承認が必要
- audit.md に全ユーザー入力を完全な形で記録（要約禁止）
- チェックボックスは作業完了と同時に更新
- タイムスタンプはISO 8601形式（YYYY-MM-DDTHH:MM:SSZ）

## ファイル配置規約
- アプリケーションコード: ワークスペースルート
- ドキュメント: `aidlc-docs/` 配下のみ
- AI設定: `.claude/` 配下

## スキル・エージェント使用規約
- 該当するSkill/Subagentがある場合は積極的に使用
- 使用するSkill/Subagentと理由を1行で明記してから進める

## タスク完了時の確認事項
1. audit.md にログを追記
2. aidlc-state.md を更新
3. チェックボックスを更新
4. ユーザー承認を取得してから次フェーズへ
